Bank System

Welcome to Bank System! Your journey to easy and secure banking starts here. Explore our features and see how we can simplify your financial management.

Features

Main Menue Screen

The main function in the Bank System interface allows users to select an action by entering a number from 1 to 10. Each number corresponds to a specific function such as viewing the client list, adding or deleting a client, updating client information, performing transactions, managing users, logging in or out, and currency exchange.

Transactions Screen

The Transactions Menu allows users to manage financial activities. Options include depositing money, withdrawing funds, viewing total account balances, transferring money between accounts, and reviewing the transfer log. Selecting an option by entering a number from 1 to 6 performs the corresponding action or returns to the main menu.

Manage Users Screen

The Manage Users Menu allows you to handle user accounts within the Bank System. You can list all users, add new ones, delete existing users, update user information, or search for a specific user. Selecting an option by entering a number from 1 to 6 performs the chosen action or returns to the main menu..

Login Register List Screen

The Login Register List Screen shows records of user login attempts, including timestamps, usernames, passwords, and permissions. It displays data for each login event, allowing monitoring of user activities. This log helps in tracking successful and unsuccessful login attempts, highlighting any security concerns or issues with user access.

Currancy Exhange Main Screen

The Currency Exchange Menu allows users to manage currency-related tasks. Options include listing available currencies, searching for a specific currency, updating exchange rates, and using a currency calculator for conversions. Selecting an option by entering a number from 1 to 5 performs the chosen action or returns to the main menu.

Bank System: About the Project

The Bank System project was developed to offer a secure and user-friendly mobile banking solution. Its primary goal is to enable users to efficiently manage their bank accounts through functionalities such as deposits, withdrawals, transfers, and transaction history reviews. Designed as a portfolio project for Holberton School, it showcases expertise in mobile app development and financial management.
